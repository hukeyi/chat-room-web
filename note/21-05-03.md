## 在 login 界面响应 enter 键

不知道为什么，就算把`@click`删掉，`@keyup.enter`也不会响应；
并且，即使使用原生标签`<button>`，依然不会响应，于是选用直接在根 dom 上绑监听的方式

```
mounted() {
    // 响应enter键
    window.addEventListener('keyup', (event) => {
        if (event.key === 'Enter') {
            this.userId && this.password && this.handleClickLogin();
        }
    });
},
```
