## sequelized 查询多条数据

```
const friends_info = await User.findAll({
		where: {
			id: idList,
		},
		attributes: ['id', 'name', 'avator', 'gender', 'age', 'birth_date'],
	});
```

`id: idList`，其中`idList`是一个 id 的数值数组，直接传，不需要用`$in`，不知道为啥用了这个反而不起效
